🌐Browser Object Model (BOM) & Alert Boxes

Browser ke features ko JavaScript se control karo!

BOM Kya Hai? 🤔

Browser Object Model (BOM) browser ke features ko access karne ka tarika hai. Yeh JavaScript objects hain jo browser ko control karte hain!

BOM Ka Hierarchy:

🪟 Window Object (Sabse top level)
📍 Location - URL manage
📜 History - Browser history
📊 Document - HTML content
ℹ️ Navigator - Browser info
📺 Screen - Display info
⏱️ Timers - setTimeout, setInterval

💡 Note: BOM browser-specific hai. Server pe kaam nahi karta!

1️⃣ Window Object

Sabse important object! Browser ke sab features yahan se access hote hain.

Window Object Properties:

// Window object sabhi variables ka parent hai console.log(window.innerWidth); // ViewPort width console.log(window.innerHeight); // ViewPort height console.log(window.outerWidth); // Full browser width console.log(window.outerHeight); // Full browser height // Window object ke through sab access hote hain window.console.log("Hello"); // Same as console.log() window.document.title; // Same as document.title

2️⃣ ALERT BOX - Alert Dialog

User ko message dikhate ho. Sirf "OK" button hota hai.

Syntax:

alert("Message here");

Features:

✅ User ko notification dena ho
✅ Sirf "OK" button (mandatory)
✅ Return value nahi hota
✅ Webpage freeze hota hai jab tak close na karo

Examples:

// Simple alert alert("Welcome to my website!"); // Multi-line message alert("Hello User!\nYou logged in successfully.\nEnjoy your experience!"); // With variables const name = "John"; alert(`Welcome, ${name}!`); // After some action function submitForm() { alert("Form submitted successfully!"); }

🎮 Try Alert Box:

⚠️ Warning: Production websites mein excessive alerts avoid karo. User experience kharab hota hai!

3️⃣ CONFIRM BOX - Yes/No Dialog

User se confirmation lena ho. "OK" aur "Cancel" buttons hote hain.

Syntax:

const result = confirm("Are you sure?"); // Returns: true (OK) or false (Cancel)

Example:

function deleteAccount() { const confirmed = confirm("Are you sure you want to delete your account?\nThis action cannot be undone!"); if (confirmed) { alert("Account deleted successfully"); // Send delete request to server } else { alert("Account deletion cancelled"); } } // Another example - File download function downloadFile() { if (confirm("Large file (500MB). Download now?")) { // Start download console.log("Downloading..."); } else { console.log("Download cancelled"); } }

🎮 Try Confirm Box:

4️⃣ PROMPT BOX - User Input Dialog

User se input lena ho. Text field ke saath "OK" aur "Cancel" buttons.

Syntax:

const userInput = prompt("Ask something", "Default value"); // Returns: string or null

Example:

function askName() { const name = prompt("What's your name?", "John"); if (name !== null && name.trim() !== "") { alert(`Hello, ${name}!`); } else { alert("No name provided"); } } // Another example - Age verification function checkAge() { const age = prompt("Enter your age:", "18"); if (age === null) { alert("Cancelled"); } else { const ageNum = parseInt(age); if (ageNum >= 18) { alert("Access granted"); } else { alert("You must be 18+"); } } }

🎮 Try Prompt Box:

5️⃣ Location Object - URL Management

Current URL ko access aur modify karte ho.

Location Properties:

Property Description Example
location.href Complete URL https://example.com/page
location.protocol Protocol (http/https) https:
location.hostname Domain name example.com
location.pathname Path after domain /page
location.search Query parameters ?id=123&name=john

Methods:

location.reload(); // Page refresh location.replace(url); // Go to new URL (replace history) location.assign(url); // Go to new URL (add to history) location.href = "https://example.com"; // Redirect

6️⃣ History Object - Browser History

Back, forward, aur go to specific page browser history se.

Methods:

history.back(); // Go back (same as back button) history.forward(); // Go forward history.go(n); // Go to specific page (positive or negative) // Examples: history.go(-1); // Go back 1 page history.go(2); // Go forward 2 pages

7️⃣ Navigator Object - Browser Information

Browser aur device ke baare mein information.

Common Properties:

navigator.appName; // Browser name navigator.appVersion; // Browser version navigator.userAgent; // Full browser info navigator.platform; // OS (Win32, MacIntel, etc) navigator.language; // Browser language navigator.onLine; // Internet connection status navigator.geolocation; // User location navigator.cookieEnabled; // Cookies enabled or not

8️⃣ Screen Object - Display Information

Display screen ke dimensions aur colors ke baare mein.

Properties:

screen.width; // Screen width in pixels screen.height; // Screen height in pixels screen.availWidth; // Available width (excluding taskbar) screen.availHeight; // Available height screen.colorDepth; // Color depth (bits per pixel) screen.pixelDepth; // Pixel depth

9️⃣ Timers - setTimeout & setInterval

Code ko delay se ya repeatedly execute karte ho.

setTimeout - Ek baar execute:

const timerId = setTimeout(() => { console.log("This runs after 3 seconds"); }, 3000); // 3000 milliseconds = 3 seconds // Cancel timeout clearTimeout(timerId);

setInterval - Repeatedly execute:

const intervalId = setInterval(() => { console.log("This runs every 2 seconds"); }, 2000); // Stop interval clearInterval(intervalId);

Real-World Examples:

// Auto-save every 30 seconds setInterval(() => { console.log("Auto-saving..."); }, 30000); // Click button, then 2 sec baad redirect function submitForm() { alert("Form submitted!"); setTimeout(() => { location.href = "/success"; }, 2000); }

🔟 Alert Dialogs Comparison

Dialog Buttons Returns Use Case
alert() OK only - Information/notification
confirm() OK, Cancel true/false Yes/No confirmation
prompt() OK, Cancel string/null User input

🌟 Real-World Examples

Example 1: Login Validation

function login() { const username = prompt("Enter username:"); if (username === null) { alert("Login cancelled"); return; } if (username.length < 3) { alert("Username must be at least 3 chars"); return; } alert(`Welcome, ${username}!`); setTimeout(() => { location.href = "/dashboard"; }, 2000); }

Example 2: Confirm Action

function deletePost(postId) { const confirmed = confirm( `Delete post ${postId}?\nThis cannot be undone!` ); if (confirmed) { alert("Post deleted successfully"); // Delete from server } }

Example 3: Timer-based Popup

function startPopupTimer() { alert("Survey popup will appear in 5 seconds..."); setTimeout(() => { const rating = prompt( "Rate our website (1-10):", "5" ); if (rating !== null) { alert(`Thank you! You rated us ${rating}/10`); } }, 5000); }

✅ Best Practices

Kab Use Karo:

✅ Small notifications/alerts
✅ Quick confirmations
✅ Simple user input

Kab Use Na Karo:

❌ Too many alerts (annoying)
❌ Complex forms (use HTML forms instead)
❌ Auto-triggering alerts
❌ Production websites (outdated feel)

Modern Alternative:

// Instead of alert(), use custom modals/toasts function showNotification(message) { const notification = document.createElement("div"); notification.textContent = message; notification.style.cssText = ` position: fixed; top: 20px; right: 20px; background: #667eea; color: white; padding: 15px; border-radius: 4px; z-index: 9999; `; document.body.appendChild(notification); setTimeout(() => notification.remove(), 3000); }
